# Tic Tac Toe

A classic two-player Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript.

![Tic Tac Toe](https://img.shields.io/badge/HTML-CSS-JavaScript-blue)

## Features

- **Two-player gameplay** — Players alternate between X and O
- **Win detection** — Automatically detects when a player wins
- **Reset game** — Start a new game at any time
- **Responsive design** — Adapts to different screen sizes using viewport units
- **No dependencies** — Pure vanilla JavaScript, no frameworks or libraries

## How to Play

1. Open `index.html` in your web browser
2. Player X goes first
3. Click an empty cell to place your mark
4. The first player to get three marks in a row (horizontal, vertical, or diagonal) wins
5. Click **Reset Game** to start over

## Getting Started

No build step or installation required. Simply open the project:

```bash
# Option 1: Open directly in browser
start index.html   # Windows
open index.html   # macOS
xdg-open index.html   # Linux

# Option 2: Use a local server (optional)
npx serve .
```
